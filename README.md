# 🎧 疯狂听抄 - 让你不再恐惧听力

一个纯前端实现的英语听力听写练习工具，支持变速播放、自动批改、生词本、句子收藏和听力历史记录。

## ✨ 功能亮点
- 📚 **内置多类素材**：高考、四六级、托福雅思、BEC、BBC六分钟、TED-ED、科学60秒等
- ⬆️ **支持上传本地MP3**（10MB以内），自动识别文本并断句
- ⏩ **变速播放**：0.5× ~ 2× 倍速，变速不变调（浏览器TTS合成）
- 🔁 **单句循环 + 自定义遍数**，句间间隔可调
- ✅ **自动批改**：对比用户输入与原文，高亮正确/错误单词，计算准确率
- 📖 **生词本 & 句子收藏**，方便复盘
- 📋 **听力历史记录**，随时回看每次练习成绩
- ⌨️ **键盘快捷键**：空格 播放/暂停，Ctrl+→ 下一句，Ctrl+← 上一句
- 🌐 纯静态页面，可直接部署到 GitHub Pages
